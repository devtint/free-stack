Backend / API
Render

Deploy Node, Python, Go, or any Dockerised app. Free tier spins down after 15 min of inactivity — keep alive with a health-check cron.

  • 750 hrs / month free (enough for 1 always-on service)
  • Set up a cron job → ping GET /health every 5 min
  • Use cron-job.org or UptimeRobot for the ping — both free
  • Supports background workers & scheduled jobs
  • Auto-deploy from GitHub on push
  • Free TLS + custom domain
Node.js Python Docker REST
✈
Telegram Bot
Bot API (free)

Host your bot logic on Render. Use webhooks (not polling) for zero idle CPU — Telegram pushes updates directly to your endpoint.

  • Register bot free via @BotFather
  • Set webhook to your Render HTTPS URL
  • Webhook mode = no polling = no wasted compute
  • Use grammY or python-telegram-bot library
  • Store state / user data in Supabase
  • Schedule broadcast jobs with Render Cron
