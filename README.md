# 🆓 Free Stack

A visual reference card for building production-ready apps without spending a dollar.

## Live Demo
> https://your-username.github.io/free-stack

## What's Inside
A single-page HTML reference covering the best free-tier services for indie hackers and solo builders:

- **Vercel** — Frontend hosting & CDN
- **Supabase** — Postgres DB, Auth, Storage, Realtime
- **Render** — Backend API & Telegram Bot hosting
- **GitHub** — Source control, CI/CD, Pages
- **Cloudflare** — DNS, CDN, Workers, R2 Storage
- **UptimeRobot / cron-job.org** — Keep-alive pings
- **Resend** — Transactional email
- **Clerk** — Auth with social login & MFA
- **Sentry** — Error tracking
- **OneSignal** — Push notifications
- **Umami** — Self-hosted analytics

## Deploy in 30 seconds
1. Fork this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Visit `https://your-username.github.io/free-stack`
