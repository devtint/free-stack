# 🆓 Free Stack

A visual reference card for building production-ready apps without spending a dollar.

## Live Demo
> https://devtint.github.io/free-stack

## What's Inside

### Reference Guide (`index.html`)
A single-page HTML reference covering the best free-tier services for indie hackers and solo builders:

- **Vercel** — Frontend hosting & CDN
- **Supabase** — Postgres DB, Auth, Storage, Realtime
- **Render** — Backend API & Telegram Bot hosting
- **GitHub** — Source control, CI/CD, Pages
- **Cloudflare** — DNS, CDN, Workers, R2 Storage
- **UptimeRobot / cron-job.org** — Keep-alive pings
- **Resend** — Transactional email
- **Clerk** — Auth with social login & MFA
- **Sentry** — Error tracking
- **OneSignal** — Push notifications
- **Umami** — Self-hosted analytics

### Reference Guides
- **`telegram-bot-guide.html`** — Build & deploy Telegram bots on Render
- **`render-guide.html`** — Web services, keep-alive, webhooks, render.yaml on Render
- **`supabase-guide.html`** — Database, Auth, Storage, Realtime on Supabase
- **`vercel-guide.html`** — Deploy, Env Vars, Serverless Functions, Domains on Vercel
- **`cloudflare-guide.html`** — DNS, Workers, R2 storage, proxy setup on Cloudflare
- **`github-guide.html`** — Repos, Actions CI/CD, Secrets, Pages, branch protection
- **`keepalive-guide.html`** — UptimeRobot, cron-job.org, Supabase anti-pause strategies
- **`git-guide.html`** — Git CLI: setup, workflow, branching, undo, history, .gitignore (with real examples)
